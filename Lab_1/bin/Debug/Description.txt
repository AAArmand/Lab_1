Программа работает
в режиме запрос пользователя - действие.
Командам, которым требуются параметры,
необходимо передавать их введя после
самой команды через пробел.
Реализованы следующие команды:
 1) help - выводит описание программы и возможные команды.
 2) iterations - устанавливает количество повторений алгоритмов
 для вычисления среднего времени работы. Параметр для этой команды - количество итераций.
 Значение по умолчанию: 100
 3) sequence - устанавливает последовательность для тестирования,
 то есть целые числа через пробел.
 4) test - запускает работу алгоритмов.
 Каждый алгоритм выполняется на последовательности sequence, iterations раз.
 В результате выводится среднее время работы каждого алгоритма.
 5) exit - выход из программы.
